## Вступление
В качестве индивидуальных правок для предыдущего проекта я решил кардинально переделать структуру кода.
До этого в проекте были модули: main.py, preprocess.py, visualization.py, visualization_advanced.py. После моих правок в проекте остались только main и visualization_advanced.py.
Сейчас я расскажу почему я так сделал. Main.py стал точкой входа, теперь программа запускается через этот модуль. Также preprocess теперь интегрирован именно в этот модуль, тоесть предобработка проходит через него.
Также я решил перевести visualization в visualization_advanced, потому что продвинутая визуализация исключает необходимость в обычной визуализации.

## Структура
├── main.py # Функции подготовки данных
├── visualization_plotly.py # Создание визуализаций
├── melb_data.csv # Исходный датасет
└── output/ # Папка с результатами

## Требования
- Python 3.7+
- pandas
- plotly

## Визуализации
Карта медианных цен — географическое распределение цен по пригородам
Топ-20 пригородов — самые дорогие районы (≥10 продаж)
Box plot цен — распределение по типам недвижимости (дом/квартира/таунхаус)
Динамика цен — изменение медианных цен по месяцам
Цена за м² — сравнение регионов и типов недвижимости
Корреляционная матрица — взаимосвязь характеристик недвижимости
Цена vs расстояние — зависимость цены от удалённости от центра
Цена по комнатам — как меняется цена в зависимости от количества комнат
Методы продажи — анализ популярности и эффективности методов
Сводная таблица — интерактивная таблица со всеми показателями

## Использование
Скрипт автоматически:
Загрузит и подготовит данные из melb_data.csv
Создаст папку output/
Сгенерирует 10 HTML-файлов с интерактивными графиками

### Основные возможности:
1. Автоматическая подготовка данных**
- Очищает пропуски и некорректные значения
- Вычисляет дополнительные метрики: возраст дома, цену за м², плотность застройки
- Нормализует форматы дат и числовых полей
2. Географический анализ**
- Показывает, где находятся самые дорогие районы на карте
- Визуализирует зависимость цены от расстояния до центра города
- Определяет топ-20 самых дорогих пригородов
3. Сравнительный анализ**
- Сравнивает цены по типам недвижимости (дома vs квартиры vs таунхаусы)
- Анализирует цену за м² по разным регионам
- Показывает, как цена зависит от количества комнат
4. Временной анализ**
- Отслеживает динамику изменения цен по месяцам
- Выявляет тренды роста/падения для разных типов недвижимости
- Помогает определить оптимальное время для покупки/продажи
5. Корреляционный анализ**
- Показывает, какие характеристики больше всего влияют на цену
- Выявляет взаимосвязи между площадью, расположением, возрастом и ценой
6. Анализ методов продажи**
- Сравнивает эффективность разных методов (аукцион, прямая продажа и др.)
- Показывает распределение цен по методам продажи
